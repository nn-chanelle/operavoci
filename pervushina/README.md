# Voci dell'Opera — Лендинг творческой встречи с Екатериной Первушиной

Камерный одностраничный лендинг под событие 24 мая 2026.

## Что нужно сделать перед публикацией

### 1. Добавить фото Кати
Положи фото в папку `assets/` под именем `katya.jpg` (или `.png`).
Затем в `index.html` найди блок:

```html
<div class="photo">
  <!-- Замените содержимое на: <img src="assets/katya.jpg" alt="Екатерина Первушина" /> -->
  <span>Здесь будет портрет<br/>Екатерины Первушиной</span>
</div>
```

И замени на:

```html
<div class="photo">
  <img src="assets/katya.jpg" alt="Екатерина Первушина" />
</div>
```

Оптимальный размер фото — вертикальный портрет, минимум 920×1120 px, формат JPG (до 400 КБ).

### 2. Вставить ссылку на TimePad
В `index.html` найди:

```html
<a class="cta" href="#" id="ticketLink">забронировать билет</a>
```

Замени `#` на ссылку события TimePad — например:
```html
<a class="cta" href="https://operavoci.timepad.ru/event/3007654/" id="ticketLink">забронировать билет</a>
```

## Как опубликовать на GitHub Pages (бесплатно, ~10 минут)

### Шаг 1. Создать репозиторий
1. Зайти на https://github.com → New repository
2. Имя: `voci-event-pervushina` (или любое латиницей)
3. Сделать **Public**
4. Создать

### Шаг 2. Залить файлы
Самый простой способ — через веб-интерфейс GitHub:
1. На странице нового репо нажать **uploading an existing file**
2. Перетащить туда:
   - `index.html`
   - `README.md`
   - всю папку `assets/` (с logo.jpg и katya.jpg)
3. Внизу страницы — Commit changes

### Шаг 3. Включить GitHub Pages
1. В репозитории → Settings → Pages (в левом меню)
2. Source: Deploy from a branch
3. Branch: `main`, папка `/ (root)`
4. Save
5. Через 1–2 минуты GitHub покажет публичную ссылку:
   `https://<твой-логин>.github.io/voci-event-pervushina/`

Эта ссылка и идёт во все посты, в шапку Instagram, в Telegram-канал.

## Кастомный домен (опционально)

Если хочешь короткую ссылку типа `events.voci.ru`:
1. У домена прописать CNAME → `<твой-логин>.github.io`
2. В Settings → Pages → Custom domain ввести `events.voci.ru`
3. Включить Enforce HTTPS

## Что внутри лендинга

- Палитра: кремовая + бордо + золото
- Шрифты Google Fonts: **Cormorant Garamond** (заголовки), **Marck Script** (рукописные акценты), **Inter** (текст)
- Адаптивная вёрстка (мобильный + десктоп)
- Плавные появления блоков при скролле
- Open Graph метаданные для красивого превью при шаринге

Все шрифты грузятся с Google Fonts CDN — никаких внешних зависимостей кроме этого.

## Если нужно что-то поменять

Все цвета вынесены в CSS-переменные в начале файла — меняешь одну строку, обновляется везде:

```css
--cream: #faf4e8;          /* основной фон */
--bordeaux: #6b1e2a;        /* акцентный бордо */
--gold: #b89766;            /* золото */
```
